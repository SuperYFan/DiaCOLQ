# DiaCOLQ Benchmark Dataset

This repository contains the benchmark dataset associated with the paper "DiaCOLQ: A benchmark for Chinese offensive language quadruple prediction in multi-turn dialogues".

DiaCOLQ is designed for the task of Chinese offensive language quadruple prediction in multi-turn, multi-speaker dialogues. It serves as the benchmark dataset used in the paper for evaluating models on structured offensive language understanding, including target, argument, category, and speaker prediction.

If you use this dataset, please cite the corresponding paper.
